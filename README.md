# ReBrew 

Streamlit dashboard for ReBrew that:
- Shows top 5 marketing insights with clean, emojiâ€‘based charts.
- Compares Decision Tree, Random Forest and Gradient Boosting on subscription prediction.
- Includes clustering, association rules and regression labs.
- Lets you enter a new customer profile and download the full dataset with predicted labels.

## How to use

1. Create a new GitHub repo.
2. Upload these files directly to the root (no folders):
   - `app.py`
   - `ReBrew_Market_Survey_Synthetic_Data_600_responses.xlsx`
   - `requirements.txt`
   - `README.md`
3. On Streamlit Cloud, create a new app and point it to `app.py` on your main branch.
